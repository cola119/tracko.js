# tracko
Realtime GPS tracking system for orienteering.
オリエンテーリング競技中の選手の位置をリアルタイムで表示します。

# Usage
You need to install trako(iOS app) and upload orienteering maps.
in more detail:
https://cola119.hatenablog.com/entry/2018/12/15/233344
